the SPICE applet

What is the SPICE applet ?

The SPICE applet allows to visualize protein structures and their annotations, based on the DAS protocol. DAS, the distributed annotation system, provides a communication protocol to exchange biological annotations. It is based on the idea that annotations should not be provided by single centralized databases, but rather be spread over multiple sites. Data distribution, performed by DAS servers, is separated from the visualization, which is done by DAS clients. SPICE is such a client which connects to different servers - perhaps even you own!

SPICE is based on the extensions to the DAS protocol as defined at http://www.sanger.ac.uk/xml/das/documentation/. For the 3D visualization the Jmol applet is used.

some example DAS queries

Extract the protein structure 3D coordinates for the protein structure with PDB code 1a4a.

  • http://das.sanger.ac.uk/das/structure/structure?query=1a4a

Get the alignment to Uniprot for 1a4a:

  • http://das.sanger.ac.uk/das/aligpdbsp/alignment?query=1a4a

Get the Interpro features of seqeunce P00280

  • http://das.sanger.ac.uk/das/interpro/features?segment=P00280

Where can I download the sourcecode of SPICE?

The SPICE source code is available (soon) under LGPL using Subversion from http://www.derkholm.net/svn/repos/spice/trunk/. Just do 

svn checkout http://www.derkholm.net/svn/repos/spice/trunk/ spice

You will also need to check out Biojava and Jmol projects.

Thanks to the Jmol and Biojava projects!
