Run SPICE

There are two ways to run SPICE:

  • as a Java applet
  • using Java Web Start

Both approaches are described below.

SPICE as a Java Applet


Press button to start SPICE

The applet automatically loads PDB 1a4a. The applet provides an entry field to display other structures, for which 3D coordiantes and SwissProt-PDB mappings are available.

You might want to open the Java Console of your browse to see more details about the DAS communication that is going on in the background.

The applet has been tested successfully using java 1.4, Mozilla, Firefox, and IE6 under Linux, OSX and Windows XP operating systems.

  • Java 1.4 on OSX: If you have troubles running the applet under OSX: Download the replacement Java plugin from: http://javaplugin.sourceforge.net/ and follow the instructions in the Readme.txt file.

SPICE and Java Web Start

Using Java Web Start, software applications can be deployed with a single click over the Internet. Web Start ensures the most current version of the application will be deployed. In contrast to Java applets, where Jar files need to be downloaded every time the Applet is started, using Web Start the application is downloaded only the first time it is requested. From the second call of the Application the execution if much faster.

Press here to start SPICE using Web Start:
Run Spice !

If pressing the above link does not launch Web Start, you need to install it first. Please read more about installation of Web Start at http://java.sun.com/products/javawebstart/1.2/docs/installguide.html
